<a name="english"></a>
## English

A desktop application to easily manage multiple Java versions on Windows.

### Features

- Automatically detects existing Java installations
- Installs new versions of OpenJDK
- Switch between Java versions with a single click
- Uninstall versions you no longer need
- Manages environment variables (JAVA_HOME, PATH) automatically
- Supports multiple languages: English, Spanish, French, German, Portuguese

### Screenshots

![Main screenshot](screenshots/main.png)

### Installation

#### Option 1: Installer
1. Download the installer from the [Releases](https://github.com/moisesmagna/cup-control/releases) section
2. Run the downloaded `.exe` or `.msi` file
3. Follow the installer instructions

#### Option 2: Portable version
1. Download the ZIP file from the [Releases](https://github.com/moisesmagna/cup-control/releases) section
2. Extract the contents to any location on your system
3. Run `Cup Control.exe`

### Usage

1. When you start the application, you'll see the Java versions installed on your system
2. To install a new version, select it from the dropdown menu and click "Install"
3. To activate a version, select it and click "Activate"
4. To uninstall a version, click "Uninstall"

### Development

#### Prerequisites
- Node.js 18 or higher
- npm or yarn

#### Development setup
```bash
# Clone the repository
git clone https://github.com/moisesmagna/cup-control.git
cd cup-control

# Install dependencies
npm install

# Start in development mode
npm run electron:dev
```
